# appm-leyu-hub

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定网站或域名，而是作为通用前端资源存储与展示的集合。

## 项目简介

appm-leyu-hub 是一个静态 HTML 页面归档仓库。仓库内收录了若干独立网页文件，每个文件均为自包含的 HTML 页面，可用于演示、参考或轻量级功能展示。本仓库不包含后端逻辑、数据库或外部依赖，所有页面均可直接通过浏览器打开查看。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 主要页面归档目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面文件。
- `assets/`：存放页面引用的公共资源，如样式表、JavaScript 脚本和图片等。
- `index.html`：可选，可作为仓库首页或导航入口。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，部分页面可能引用 `assets/` 目录下的资源。
- 页面之间相互独立，没有数据共享或依赖关系。
- 归档页面不针对任何具体网站或服务，仅作为通用前端页面展示。
- 页面内容可能包含示例数据、占位文本或演示功能，不构成实际服务或产品承诺。
